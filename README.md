# go_poker

## How to use
### Clone and change dir
```
$ git clone https://github.com/klein-mask/go_poker.git
$ cd go_poker
```

### Default play
```
$ go run main.go
```

### Custom 「BigBlind」, 「SmallBlind」 and 「Player Money」

**example**
- BigBlind: 600
- SmallBlind: 300
- Player Money: 20000

```
$ go run main.go 600 300 20000
```
